Implement the requested change in the current repository.

Constraints:
- Follow repository instructions from AGENTS.md, CLAUDE.md, .claude, and .codex resources.
- Keep changes scoped to the request.
- Preserve unrelated user changes.
- Verify with the narrowest useful command.
- Before reporting, review your own diff for correctness, unnecessary complexity, and missing validation.
